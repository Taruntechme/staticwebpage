authentication is the act of providing security option to any website, applicationby the system user. it means without the security option no one can access their application and website. we can say that it is the process of verifying the system user identity.

If we want to take example it might involve personal identity documents,verifying the authenticity of a website with digital certificate a common problem is verifying that a given artifact was produced by a certain person or in a certain place or period of history.A system verifying a user's identity is often required to allow access to confidential data or systems.


Authentication can be considered to be of three types:
 In first type of authentication include :-
  • he knowledge factors: Something the user knows ( Example :- a password, partial password, pass phrase, or personal identification number (PIN), challenge response (the user must answer a question, or pattern), Security questionauthentication can be a proof of identity given by a credible person who has first-hand evidence that the identity is genuine.
  • When authentication is required of art or physical objects, this proof could be a friend, family member or colleague attesting to the item's provenance, perhaps by having witnessed the item in its creator's possession.
    • In Second type of authentication include :-
  • the ownership factors: Something the user has ( wrist band, ID card, security token, implanted device, cell phone with built-in hardware token, software token, or cell phone holding a software token) Authentication is comparing the attributes of the object itself to what is known about objects of that origin.
    For example, An art expert might look for similarities in the style of painting, check the location and form of a signature, or compare the object to an old photograph.
    • In Third type of authentication include :-
  • Inherence factors : Something the user do ( fingerprint, retinal pattern, DNA sequence (there are assorted definitions of what is sufficient), signature, face, voice, unique bio-electric signals, or other biometric identifier.
